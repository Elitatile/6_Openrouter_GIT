AI ASSISTANT - TEXT SUMMARIZER & EMAIL RESPONSE GENERATOR
Quick Start Instructions

STEP 1: INSTALL DEPENDENCIES
================================================================================
Ensure Python 3.8+ is installed, then run:

    pip install streamlit openai


STEP 2: SET UP API KEY
================================================================================
1. Get your OpenRouter API key from: https://openrouter.ai
2. Create a file named "openrouter_api_key.txt" in this project directory
3. Paste your API key (just the key, nothing else)
4. Save the file

Example (macOS/Linux):
    echo "your_api_key_here" > openrouter_api_key.txt

Example (Windows PowerShell):
    "your_api_key_here" | Out-File -Encoding UTF8 openrouter_api_key.txt


STEP 3: RUN THE APPLICATION
================================================================================
From the project directory, run:

    streamlit run app.py

The app will automatically open in your browser at:
    http://localhost:8501

TROUBLESHOOTING
================================================================================

Problem: "openrouter_api_key.txt not found"
Solution: Create the file with your API key - see STEP 2

Problem: "openrouter_api_key.txt is empty"
Solution: The file exists but has no API key. Add your key and save.

Problem: "Error generating summary"
Possible causes:
  - Invalid or expired API key
  - No internet connection
  - Insufficient API credits on OpenRouter
  - Model temporarily unavailable
Solution: Verify your API key at https://openrouter.ai and check your balance

Problem: Browser not opening automatically
Solution: Manually open http://localhost:8501 in your web browser

Problem: "Module not found" error
Solution: Run: pip install streamlit openai
